## 우아한 테크코스 프리코스 1주차 미션 - 숫자 야구 게임

### 구현 기능 목록

* 서로 다른 임의의 수 3개 구해서 세팅하기

* 입력 받기
    * 사용자 숫자 입력 시 예외상황 처리
        * 3자리 숫자인지
        * 숫자 이외의 문자
        * 중복된 수가 있는지
	
    * 메뉴 숫자 입력 시 예외상황 처리
         * 1 또는 2인지
        * 숫자 이외의 문자

* 출력하기
    * 힌트 출력하기
    * 메뉴 출력하기
    * 입력 문구 출력하기
    * 정답 문구 출력하기

* 힌트 계산하기

* 정답인지 확인하기

* 게임 진행 로직 구현
